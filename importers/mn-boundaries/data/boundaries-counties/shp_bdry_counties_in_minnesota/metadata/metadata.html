Section 1 Overview
Originator Minnesota Department of Natural Resources (MNDNR)
Title County Boundaries, Minnesota
Abstract This is the standard Minnesota State County Boundary dataset that is used by MNDNR and many other state agencies. It is maintained by the MNDNR Lands and Minerals Division.

Please read at the accuracy and lineage sections of this metadata to make sure this dataset is appropriate for your application!
Purpose This data was created to provide a standard county boundary layer to represent the state. This boundary dataset is NOT a legal document, nor is it appropriate for use on large scale or site-specific maps and applications. Best used at 1:24,000 or smaller scales. For more detailed and precise data we advise you contact the specific county of interest.

Section 2 Data Quality
Attribute Accuracy Attributes have been verified and tested under wide use for decades
Logical Consistency Polygon(s) are coincident, verified closed, and intersect where intended.
Completeness All Minnesota county lands are present, including off-shore islands in Lake Superior. The official state and international borders within Lake Superior are not included in this data set.
Horizontal Positional Accuracy The primary full-resolution feature class in the geodatabase is called: mn_county_boundaries

There are several reduced-resolution feature classes also included in the geodatabase which are useful for fast visual display or to support generating web services designed to be displayed at small scales.

The naming convention for these reduced-resoultion feature classes indicates the snapping tolerance that was applied. For example, vertices within the feature class mn_county_boundaries_1500 are a minimum of 1500 meters apart. Use this information to determine the appropriate feature class to use at various scales for your specific application.

When in doubt, always use the full resolution feature class by default: mn_county_boundaries

Source accuracies vary greatly. Public land survey derived lines and coincident county boundaries range in expected offsets from 50 to 200 feet, with the smaller value being more common. The most significant offsets within this category are in the northeastern portion of the state where the U.S. Geological Survey (USGS) quad sources are most tentative with regards to section corner locations. County boundaries not coincident with PLS lines are typically from 1990 vintage TIGER data with many off-sets in the 100 foot range. The international boundary should be highly accurate as it was taken directly from legislative notes. The Lake Superior North Shore was from 1980's vintage National Wetland Inventory and will typically exhibit off-sets of about 60 feet, with generally good depictions of the shore line character.

This data is NOT to be used as a legal document, and is not accurate enough for site-level analysis.
Vertical Positional Accuracy Not Applicable
Lineage Attribute Lineage: Some attribute table restructuring was performing during MNDNR's GIS Core Library development effort in March 1996. This entailed bringing in selected fields from the county cross-reference table, including county names and county abbreviations.

A simplified, multipart version of this dataset, called mn_county_boundaries_multipart, was created in July of 2013 to meet various DNR enterprise needs. The feature class has only 87 records (multipart shapes for counties with islands) and includes state-standard County FIPS codes.

A multitude of source dates and data are associated with this data set, including those associated with the original GISMO, RKI, DLG, and TIGER data set development efforts. Work on the coverage was completed during the summer/fall of 1993. Attribute restructuring to ensure conformance with MNDNR Public Land Survey (PLS) Coding standard performed in March 1996.
